# Python Development

---

## uv

### What it does

Modern Python package manager.

Replaces:

```text
pip
virtualenv
pip-tools
poetry (partially)
```

### Core Commands

Create project:

```bash
uv init
```

Create venv:

```bash
uv venv
```

Install package:

```bash
uv add fastapi
```

Run:

```bash
uv run main.py
```

Sync:

```bash
uv sync
```
